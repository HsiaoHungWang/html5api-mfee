HTML5 APIs

表單驗證

HTML5 提供了一些內建的驗證屬性，瀏覽器針對這些驗證，提供了預設的驗證訊息及訊息顯示的方式，這裡的練習主要是學習如何自訂錯誤訊息及顯示的方式

內建表單驗證設定 自訂表單驗證練習
https://html.spec.whatwg.org/multipage/forms.html#forms
<audio>

在網頁上播放音樂可以使用<audio>，也可以透過Audio API控制音樂的播放

Audio API 播放音樂
https://html.spec.whatwg.org/multipage/media.html#the-audio-element
<video>

在網頁上播放影片可以使用<vedio>標籤，它有跟Audio標籤一樣的API

影片播放
https://html.spec.whatwg.org/multipage/media.html#the-video-element
Canvas

要在網頁上繪圖就是要使用<canvas>，canvas 標籤就像是將圖紙準備好，然後透過繪圖物件，在圖紙上繪圖

基本圖形 長條圖 小畫家
https://html.spec.whatwg.org/multipage/canvas.html#the-canvas-element
SVG

SVG（Scalable Vector Graphics），可以使用標籤的方式在網頁上繪製向量圖形，圖形在放大縮小時不會失真。

基本圖形 長條圖
https://www.w3.org/Graphics/SVG/
Drag And Drop

當使用者滑鼠點選某個標籤時，可以透過拖曳的方式移動標籤，在移動到某個規劃好的區域時，可以放開滑鼠，將此標籤移動到此區域

拖曳及放下 加入購物車
https://html.spec.whatwg.org/multipage/dnd.html#dnd
File And FileReader

使用者選取了檔案後，可以透過 File API 用來取得檔案相關資訊，透過 FileReader API 取得檔案內容

File API 讀取文字 讀取圖片
https://www.w3.org/TR/FileAPI/
Web Storage

在瀏覽器中除了可以透過 cookies 儲存一些資料外，也可以透過 Web storage 來儲存

localStorage vs sessionStorage 便利貼
https://html.spec.whatwg.org/multipage/#toc-webstorage
Server Sent Event

Server Sent Event 是一種單向通訊的技術，讓瀏覽器可以單方向的收到伺服器傳送過來的資料

EventSource Youbike即時資訊
https://html.spec.whatwg.org/multipage/server-sent-events.html#server-sent-events
WebSocket

WebSocket 是一種網路雙向的傳輸協定，常用來作線上遊戲、聊天室等功能。

WebSocket API 聊天室
https://websockets.spec.whatwg.org/
Clipboard

用來讀取與寫入剪貼簿的 JavaScript API

Clipboard API
https://www.w3.org/TR/clipboard-apis/
Notification

Notification 透過瀏覽器發出訊息，透過使用者電腦系統的桌面通知來顯示訊息。

Notification API
https://notifications.spec.whatwg.org/
Geolocation

Geolocation 主要是透過瀏覽器，取得上網裝置所在位置的經緯度，常搭配 Google Map 或 OpenStreetMap 等地圖圖資一起使用。

Geolocation API
https://www.w3.org/TR/geolocation/
Workers

瀏覽器中的JavaScript程式如果需要執行比較久時，網頁的操作會被卡住，因此需要長時間執行的程式，會放到 Worker 中執行。

Web workers SharedWorker1 SharedWorker2
https://html.spec.whatwg.org/multipage/#toc-workers
啟動圖示與離線應用

透過 manifest.json 的宣告，可以模擬出跟原生應用程式一樣，在桌面上出現啟動應用程式的圖示

在Service Workers中，透過Cache Storage來儲存靜態檔案，當瀏覽器中要求檔案時，如果Cache Storage中有檔案會直接回傳

離線應用
https://www.w3.org/TR/appmanifest/ https://www.w3.org/TR/service-workers/
